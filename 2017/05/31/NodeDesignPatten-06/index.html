Factory

在 Javascript 中 設計通常以可用，簡單並且模組化開發為主旨

而使用 Factory 則可以在物件導向上包覆一層 Wrapper

使用上可以更加彈性

在 Factory 中可以透過 Object.create() 來建立一個物件

或是利用特定的條件來建立不同的物件

組合工廠

利用遊戲的方式來解說

通常一種遊戲會有多種角色

每種角色會擁有各自不同的基本能力

  • Runner: 可移動
  • Samurai: 可移動並且攻擊
  • Sniper: 可射擊但不可移動
  • Gunslinger: 可移動並且射擊
  • Western Samurai: 可以移動 攻擊 並射擊

希望上述可以自由地互相結合

所以不能使用 Class 或是 inheritance 來解決
